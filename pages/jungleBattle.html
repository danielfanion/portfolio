Jungle Battle

Projet Universitaire (Semestre 4)

Description du projet

En binôme, nous avons réalisé un jeu web. Le jeu reprend le principe d'Agar.io, nous avons choisi de faire une charte graphique dans le thème de la jungle, d'où le nom.
Ce projet nous a permis de découvrir les web sockets et l'utilisation des Canvas.

Compétences développées

  • JavaScript
  • TypeScript
  • Node.js
  • GIT
  • Travail en équipe
