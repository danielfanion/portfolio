Projet Universitaire (Semestre 3)

Description du projet

Scénario: Une entreprise de développement logiciel souhaite améliorer les modes de communication à la fois des collaborateurs entre eux, mais également entre les collaborateurs et les clients de l’entreprise (notamment à des fins de support). En binôme, nous avons donc réalisé le déploiement d'une application de communication (matrix) sur un serveur Synapse.

Compétences développées

  • VirtualBox
  • Système Unix
  • MarkDown
