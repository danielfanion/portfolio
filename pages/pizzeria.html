Pizzaland - API REST

Projet Universitaire (Semestre 4)

Description du projet

En binome, nous avons réalisé, pour une pizzeria fictive, une API REST pour assurer la gestion des pizzas, des ingrédients et des commandes. Nous avons réalisé le code avec un modèle MVC, en JAVA et en J2EE (Servlet). Nous utilisons une base de données Postgresql pour stocker les données. Les requêtes (hors GET) sont sécurisées avec un token que seul les admins peuvent obtenir.

Compétences développées

  • Java
  • J2EE
  • Postgresql
  • GIT
  • API Rest
  • Travail en équipe
